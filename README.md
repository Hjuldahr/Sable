# Sable
A Dynamic Discord Chat Agent
## Tech Stack
- Async SQLite (aiosqlite)
- Discord API (discord)
- Local Quantized Mistral LLM (mistral-7b-instruct-v0.1.Q4_K_M.gguf)
- Python3 (python 3.12.*)

LLM not bundled with repo: [Download](https://huggingface.co/TheBloke/Mistral-7B-Instruct-v0.1-GGUF/blob/main/mistral-7b-instruct-v0.1.Q4_K_M.gguf)
